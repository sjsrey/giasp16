Exercise 03

Date Thu 25 February 2016

Instructions

This exercises uses the following data sets that you need to have installed locally:

  • nat.geojsonnj
  • NAT.shp
  • NAT.dbf

Note you can download a zip file with the data here. Once you download the zip file and extract the files they can be found in:

giasp16/content/pages/notebooks/data

You may also want to revisit some of the notebooks from previous sessions to help carry out the following tasks.

Tasks

  1. Using the national shapefile file, construct the following spatial weights matrices:
    • Rook contiguity
    • Queen contiguity
    • knn (k=5)
  2. For the homicide rates in 1970 and 1990 carry out a descriptive statistical analysis that reports
    • Histograms
    • Kernel density of the statistical distributions
    • Summary statistics (mean, median, standard deviation, max, min)
  3. For the same rates create the following choropleth maps
    • Quintiles
    • Quartiles
    • Maximum Breaks
    • HeadsTails
  4. Using the three spatial weights matrices you built, carry out tests for global spatial autocorrelation
    • Join counts (binary based on above/below the median)
    • Moran's I

Be sure to include an interpretation of each of your analyses within the notebook.

Submission
