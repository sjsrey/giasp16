Exercise 02

Date Thu 28 January 2016

Instructions

This exercise uses the data sets introduced in class. Make sure you have them stored locally.

  1. For the Mexican income data mexico.csv, find the following for each year:
    1. Wealthiest state (name and value)
    2. Poorest state (name and value)
    3. Average state income (value)
    4. Number of states below the average income each year
  2. For the geojson file nat.json
    1. Determine how many features are in the collection
    2. Find the geometry that has the eastern most point in the collection
    3. Find the geometry that has the western most point in the collection
    4. Find the geometry that has the southern most point in the collection
    5. Find the geometry that has the northern most point in the collection
    6. Find the county that has the highest homicide rate in 1990
